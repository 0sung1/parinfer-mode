#+TITLE: parinfer-mode
#+AUTHOR: Edward Paget

This package provides a [[http://shaunlebron.github.io/parinfer/index.html][parinfer]] mode for Emacs

* Usage

- Clone this repo
- open =parinfer-mode.el= in emacs
- eval the file
- run =M-x parinfer-mode= to activate it for a buffer
